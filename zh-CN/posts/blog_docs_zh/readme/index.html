Astro 博客

一个基于 Astro、Vue 3 和 Tailwind CSS 构建的现代化、功能丰富的博客系统。

为什么选择这个博客？
  • 快速 - 静态站点生成，页面加载近乎即时
  • 现代 - 采用最新的 Web 技术构建
  • 灵活 - 支持 Markdown、MDX 和丰富的组件
  • 美观 - 简洁设计，支持深色模式

快速开始

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

然后访问 http://localhost:4321 查看你的博客！
