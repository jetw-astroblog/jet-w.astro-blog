写作指南

学习如何为你的博客创作精美、引人入胜的内容。

内容指南

指南描述
容器提示框、警告框和标注
代码块语法高亮和功能
Mermaid 图表流程图、时序图等

视频嵌入YouTube、Bilibili、自定义视频
幻灯片使用 Reveal.js 创建交互式演示文稿

Markdown 基础

Markdown 是一种轻量级标记语言，让格式化内容的编写变得简单直观。

标题

使用 # 符号表示标题：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
最佳实践
  • 每页只使用一个 # 一级标题（你的标题）
  • 不要跳级（不要从 ## 跳到 ####）
  • 保持标题简洁

文本格式

基本样式

语法结果用途
**粗体**粗体重要术语
*斜体*斜体强调
~~删除线~~删除线已弃用的内容
`代码`代码行内代码

组合样式

***粗斜体***
**粗体中有 `代码`**
